1. 保存当前工作状态
   ```
   git stash 
   ```
   恢复工作状态
   ```
   git stash pop
   ```
2. 查看版本库.git所在的目录
   ```
   git rev-parse --git-dir
   ```
3. 从一个git仓库迁移到另一个

   ```
   git clone --bare git://github.com/username/project.git  // 从原地址克隆一份裸版本库（无工作区）
   cd project.git
   git push --mirror git@gitcafe.com/username/newproject.git
   ```
